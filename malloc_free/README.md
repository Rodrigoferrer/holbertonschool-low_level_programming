# Malloc y Free en C

Este proyecto demuestra el uso de malloc() y free() para manejo de memoria dinámica en C.

## Descripción

Malloc() y free() son funciones cruciales para la gestión de memoria en C. Este proyecto incluye ejemplos prácticos de:

- Asignación de memoria con malloc()
- Liberación de memoria con free()
- Manejo de errores comunes
- Buenas prácticas de uso

## Archivos

- `malloc_example.c`: Ejemplos básicos de uso de malloc y free
- `dynamic_array.c`: Implementación de un array dinámico
- `memory_leaks.c`: Demostración de fugas de memoria y cómo evitarlas
